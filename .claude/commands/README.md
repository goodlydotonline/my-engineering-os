# .claude/commands

## Purpose
Custom slash commands for Claude Code in this repository.

## What Belongs Here
- JSON or markdown definitions of custom slash commands.
- Command documentation and usage examples.

## What Does NOT Belong Here
- General scripts (use `automation/`).
- Secrets or environment-specific values.

## Lifecycle
1. Define a command when a repetitive Claude interaction emerges.
2. Test and refine the command.
3. Update as models and use cases evolve.

## Example Entry
- `review-pr.json`: A custom slash command to trigger a structured PR review.

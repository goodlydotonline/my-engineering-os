# knowledge/vision

## Purpose
Reference material on computer vision, image processing, and related ML systems.

## What Belongs Here
- Image preprocessing and augmentation techniques.
- Model architectures for vision tasks (e.g., CNNs, ViTs).
- Evaluation metrics and dataset notes.

## What Does NOT Belong Here
- General ML theory not specific to vision.
- Raw datasets (store those externally).

## Lifecycle
1. Document vision-specific experiments and findings.
2. Distill into reusable guides.
3. Update as the field advances.

## Example Entry
- `ocr-pipeline-tuning.md`: Notes on tuning an OCR pipeline for document scanning.

# rules/prompts

## Purpose
Reusable AI prompts and prompt templates for consistent and effective AI assistance.

## What Belongs Here
- Standardized prompt templates for code review, refactoring, and documentation.
- System prompt fragments for AI workflows.
- Prompt evaluation and versioning notes.

## What Does NOT Belong Here
- AI workflow definitions (use `ai-workflow/`).
- General coding standards (use `coding/`).

## Lifecycle
1. Draft a prompt based on a recurring AI use case.
2. Test and refine for consistency and quality.
3. Version and update as models and needs change.

## Example Entry
- `code-review-prompt.md`: A standardized prompt for requesting AI-assisted code reviews.
